# Hardware + Software Revenue Growth

Interactive visualization of revenue and employee growth trajectories for 39 combined hardware/software businesses, including NVIDIA, Tesla, SpaceX, Lumafield, and more.

## Live app

**https://[username].github.io/Benchmarking/**

(Replace `[username]` with your GitHub username.)

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
```

Output is in the `dist/` folder.

## Deploy

The app deploys automatically to GitHub Pages when you push to the `main` branch. Configure the repo:

1. **Settings → Pages** → Source: **GitHub Actions**
2. Push to `main` and the workflow will build and deploy
